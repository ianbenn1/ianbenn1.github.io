Windows Command Line File Creator

In Linux based operating systems, creating a new file of any type is as simple as right clicking and naming your file. On windows, its a more irritating process. I decided to simplify file creation on windows. This is a python program that simply prompts a user for a file name/path and creates it exactly as entered. This project was made much more for my own satisfaction than for learning a language or demopnstrating skill.

Bing Canada Background Scraper

This program is a simple application for downloading the daily background of Bing Canada. I found it difficult to manually download the backgrounds from bing, as the links are deeply burried in the html. Because of this I decided to write a program to do this for me. The program opens a Bing control page XML and scrapes it for the URL tag. It then coppies the following line and downloads the picture at that address as a .jpg file labeled by date. It currently accesses only the Canadian Bing page, as that is my nationality, but this is simply altered by changing the market (mkt) tag in the XML page url.
